# RK Confeitaria

Projeto finalizado de RK Confeitaria feito com **Angular/Ionic**.

## Descrição

Este projeto é um sistema de front-end para uma confeitaria, com interface para pedidos, produtos e clientes.

## Tecnologias usadas

- Angular
- Ionic
- Capacitor
- TypeScript
- HTML/CSS

## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/Sales1998/rk-confeitaria.git

Entre na pasta do projeto:

cd rk-confeitaria


Instale as dependências:

npm install


Rode o projeto:

ionic serve
